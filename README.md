# PPPy
An implementation of the **P**erformance **P**rofile in **Py**thon.

## Usage
### Installation
```
pip3 install git+https://github.com/sakai-hiroyuki/PPPy.git
```

```
from pppy import performance_profile


performance_profile('XXX.csv')
```
